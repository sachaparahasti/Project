# Hasti Patel - Portfolio Website

Modern full-stack personal portfolio with enquiry management.

### Tech Stack
- **Frontend**: React.js, React Router, Axios, Tailwind CSS, React Toastify
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Form Validation**: Zod

### Features
- Home, About, Projects, Contact & Enquiries pages
- Responsive design (mobile + desktop)
- Enquiry form with validation & toast messages
- Real-time enquiries list from MySQL database
- Custom 404 page
